# 平台格式规范

## 公众号

### 基本参数
- 字数：1500-3000 字
- 风格：简约文艺，有深度
- 语气：真诚、有思考、不装

### 结构模板

```
标题（15字以内，有吸引力）

开头（100-200字）
→ 用一个场景/故事/问题切入，不要"今天我想分享"这种开头

正文主体（1200-2500字）
→ 观点 + 故事 + 金句，交替推进
→ 每个大观点用分割线或空行分隔
→ 段落短一些，3-5句一段
→ 适当用粗体强调关键句

结尾（100-200字）
→ 回扣开头，或留下余韵
→ 不要"总之"、"综上"这种总结式结尾
→ 可以是一个开放性问题，或一句让人想截图的话
```

### 写作要点
- 标题不要标题党，但要有点击欲
- 第一段必须抓住人，3秒决定去留
- 金句密度：每500字至少1句值得截图的话
- 故事要具体，不要泛泛而谈
- 观点要鲜明，不要面面俱到
- 结尾要留白，不要说教

### 排版规则（后续 Web UI 阶段实现）
- 简约文艺风格
- 衬线字体（如思源宋体）
- 段间距适中，留白舒适
- 重点句加粗
- 分割线用简约线条
- 配色：黑白灰为主，点缀一个暖色

## 小红书

### 基本参数
- 字数：300-800 字
- 风格：轻松、生活化、有共鸣
- 语气：像和朋友聊天，口语化

### 结构模板

```
标题（20字以内，可用 emoji）
→ 格式：emoji + 关键词 + 吸引点
→ 示例："☕ 咖啡入门｜喝了3年才明白的5件事"

正文（300-800字）
→ 开头：一句话抓住注意力
→ 中间：分点或分段，每段短小精悍
→ 结尾：引导互动（"你们觉得呢？"）

标签（5-10个）
→ 2-3个大流量标签（如 #读书 #生活感悟）
→ 3-5个精准标签（如 #认知觉醒 #独处时光）
→ 1-2个长尾标签（如 #2025读书清单）
```

### 写作要点
- 口语化！像发朋友圈，不像写作文
- 多用短句，少用长句
- 适当用 emoji，但不要满屏都是
- 分点写（1. 2. 3.），阅读体验好
- 标题要有"点进去看看"的冲动
- 结尾引导评论，提升互动

### 标签策略
- 先写大类标签（流量入口）
- 再写精准标签（目标人群）
- 最后写长尾标签（竞争小，精准触达）
- 总数 5-10 个，不要太多

### 封面文案建议（后续模板阶段实现）
- 标题大字居中
- 背景简洁（纯色或模糊图）
- 关键词突出
- 风格：简约文艺

## 两个平台的核心差异

| 维度 | 公众号 | 小红书 |
|------|--------|--------|
| 读者心态 | 有耐心，愿意读长文 | 快速浏览，3秒决定 |
| 内容深度 | 深入、完整、有观点 | 轻量、实用、有共鸣 |
| 语气 | 像写信给朋友 | 像发朋友圈 |
| 结构 | 自由叙事 | 分点、分段、清晰 |
| 标题 | 有深度，不标题党 | 有冲击力，带 emoji |
| 结尾 | 留白、余韵 | 引导互动 |
| 标签 | 不需要 | 核心流量入口 |
